# dandelion
Dandelion Desktop
A simple lightweight "Desktop" made in metacity

## Supported Distros
Ubuntu or Debian
Alpine

## Install
### Ubuntu
$ sudo apt install ruby
$ git clone https://github.com/mioojo/dandelion
$ cd dandelion
$ ruby install.rb

### Alpine
$ apk add ruby
$ git clone https://github.com/mioojo/dandelion
$ cd dandelion
$ ruby install.rb
